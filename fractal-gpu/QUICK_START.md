# 🚀 快速开始指南

## 5分钟内部署你的牛顿分形网站

### 方法1: 一键GitHub部署 ⭐（推荐）

#### 步骤1: 创建GitHub仓库
1. 登录GitHub
2. 点击右上角的 "+" → "New repository"
3. 命名仓库（例如: `fractal-gpu`）
4. 选择 "Public"
5. 不要勾选 "Add a README"
6. 点击 "Create repository"

#### 步骤2: 上传文件
1. 在仓库页面点击 "uploading an existing file"
2. 拖拽或选择所有项目文件：
   - `index.html`
   - `main.js`
   - `parser.js`
   - `index.js`
   - `shaders/` 文件夹（包含 `fractal.frag`）
3. 点击 "Commit changes"

#### 步骤3: 启用GitHub Pages
1. 进入仓库的 "Settings" 标签
2. 左侧菜单找到 "Pages"
3. 在 "Source" 部分：
   - 选择 "Deploy from a branch"
   - 选择 "main" 分支
   - 选择 "/ (root)" 目录
4. 点击 "Save"
5. 等待1-2分钟
6. 你的站点将在 `https://yourusername.github.io/fractal-gpu/` 可用

### 方法2: 本地测试（开发用）

#### 使用Python服务器
```bash
# 在项目目录中运行
python3 simple-server.py

# 或者使用内置服务器
python3 -m http.server 8080
```
访问: `http://localhost:8080`

#### 使用Node.js
```bash
# 安装依赖（可选）
npm install

# 启动开发服务器
npm run dev
```
访问: `http://localhost:3000`

## 🎮 使用方法

### 基本操作
1. **选择函数类型**: 从下拉菜单选择"多项式"或超越函数
2. **设置参数**: 
   - 多项式：输入系数（如 "1,0,0,-1"）
   - 超越函数：设置常数项
3. **调整渲染参数**: 
   - 最大迭代次数：建议30-100
   - 收敛容忍度：建议0.001-0.01
   - 颜色方案：选择喜欢的配色
4. **点击"重新渲染"**: 生成分形图案

### 交互探索
- **拖拽**: 鼠标按住拖拽平移视图
- **缩放**: 滚轮放大/缩小
- **触摸**: 在移动设备上使用手势

### 预设分形
点击预设按钮快速体验经典分形：
- z³ - 1: 经典三次多项式
- z⁴ - 1: 四次对称分形
- sin(z) - 1: 正弦函数分形
- e^z - 1: 指数函数分形

### 导出图像
点击"导出为PNG"保存当前视图

## 🎯 推荐配置

### 最佳体验
- **迭代次数**: 50-100
- **容忍度**: 0.001
- **颜色方案**: 经典彩虹
- **浏览器**: Chrome 90+, Firefox 85+, Safari 15+

### 性能优化
如果渲染慢，可以：
- 降低迭代次数到30-50
- 增大容忍度到0.01
- 使用简单颜色方案
- 缩小浏览器窗口

## 🐛 故障排除

### 问题1: 页面空白
**原因**: WebGL2不支持  
**解决**: 
- 使用Chrome 80+, Firefox 75+, Safari 14+
- 检查浏览器设置中WebGL是否启用
- 更新显卡驱动

### 问题2: 着色器错误
**原因**: 着色器编译失败  
**解决**: 
- 检查浏览器控制台（F12）
- 确保使用HTTP服务器，不要直接打开HTML文件
- 刷新页面重试

### 问题3: 渲染很慢
**原因**: 性能问题  
**解决**: 
- 降低迭代次数
- 使用较小的浏览器窗口
- 关闭其他标签页
- 检查GPU是否启用

### 问题4: 无法交互
**原因**: JavaScript错误  
**解决**: 
- 刷新页面
- 清除浏览器缓存
- 检查控制台错误信息

## 🎨 创作建议

### 探索不同函数
尝试这些有趣的函数：
